Steps for executing the program
1. First type run and press Enter
2. Inorder to store, get and count url use syntax like (1.for store url-->storeurl <example.com> 2.for get url --> get <example.com> 3. for count the no of urls --> count <example.com>
(Note: Don't add http protocol and www in the domain name simply provide like <example.com>)
Inorder to list all the urls and count use 'list' command.
Inorder to represent the arraylist value in tabular format use 'table_format' command.
